I used to be anti-JS in my high-school days, but jQuery, ES2015 and React started to change my opinion.

Joining the Insights team at Numerator increased my appreciation for React and JavaScript, particularly while I was Senior Director, with the Frontend Team Lead reporting to me (and having to fill that Team Lead role while we hired one).

At Seek, I worked with FE engineers on a React/TS + Next.js UI using Radix UI components.

Since the webpack days, TypeScript has arrived and build tools like CRA and vite have made bundling JS less painful. The React/TS + Python BE stack is very popular these days, so I still continue to take on projects that require TS. @testing-library is one of my favorite installs ever. I haven't really contributed to the OSS community for TS in the past, but I'm currently building a React/TS + Vite Frontend for a future OSS project, and I'm open to working on other TS projects should they need my help.
